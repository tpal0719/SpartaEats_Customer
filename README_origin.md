## 🍗 Sparta Eats 에서는...
- 원하는 음식을 `배달 주문`할 수 있어요
- 좋아하는 가게 있다면 `찜하기` 기능을 활용할 수 있어요
- 리뷰에 `좋아요` 를 남겨 다른 사용자들에게 유용한 정보를 제공할 수 있어요
</br>

## 📌 핵심 기능
### 사용자
- 회원 가입
- 회원 탈퇴
- 로그인
- 로그아웃
- 내 정보 조회
- 내 정보 수정
- 비밀번호 변경

### 장바구니
- 장바구니 조회
- 장바구니에 메뉴 담기
- 장바구니 메뉴 전체 삭제
- 장바구니 메뉴 단일 삭제

### 배달 주문
- 배달 주문하기
- 주문 전체 조회하기
- 주문 단일 조회하기
- 주문 삭제하기

### 가게
- 음식점 타입별 조회하기
- 음식점 상세 정보 조회하기
- 음식점 찜하기

### 메뉴
- 음식점의 메뉴 전체 확인하기
- 메뉴 세부 정보 확인하기

### 리뷰
- 주문에 대한 리뷰 작성하기
- 가게별 리뷰 조회하기
- 리뷰 상세 정보 조회하기
- 리뷰 수정하기
- 리뷰 삭제하기
- 리뷰에 좋아요 남기기
